# Caesar Cipher in Python

## Description

This project is a simple Caesar Cipher encryption and decryption program built using Python.

The program allows the user to:
- Encrypt a message
- Decrypt a message
- Choose any shift value
- Preserve spaces and special characters
- Restart the program multiple times without exiting

This project was built as part of my Python learning journey.

## Features

- Encrypt text using the Caesar Cipher algorithm
- Decrypt encrypted messages
- Handles uppercase/lowercase by converting input to lowercase
- Supports shift values greater than 26 using the modulo (`%`) operator
- Keeps spaces, numbers and punctuation unchanged
- Allows multiple encrypt/decrypt operations in one run

## Concepts Used

- Functions
- Parameters
- For loops
- While loops
- Conditional statements (`if`, `else`)
- Lists
- String methods
- Modulo operator (`%`)
- User input
- Keyword arguments

## Example

### Input

```
Direction: encode
Message: hello world
Shift: 5
```

### Output

```
mjqqt btwqi
```

## How to Run

1. Clone this repository.
2. Run the Python file.
3. Choose whether to encode or decode.
4. Enter your message.
5. Enter the shift value.
6. Repeat or exit.
